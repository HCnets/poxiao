# 南工破晓 (Poxiao) UI/UX 极致体验优化白皮书

本文档记录了自 v1.2.0 版本引入 `Liquid Glass 2.0` 及全局触觉反馈（Haptics）后，进一步向**世界一流独立应用**（如 Apple 原生应用、Things 3、Craft 等）对齐的 UI 演进路线与执行细则。

## 1. 现状剖析与对比

### 1.1 我们的绝对优势 (Pros)
1. **前卫的高定视觉质感**：相比传统大厂的扁平化（Flat Design），我们的“星辉雾晶”、“流光霓虹”结合内阴影、无级色相偏移与动态雾度，质感直逼 visionOS。
2. **物理与情感联结的微交互**：摒弃生硬的 Ripple，通过 `bouncyClick` 实现了按下微陷、松手 Q 弹的真实物理反馈，并配合多级马达震感。
3. **情境感知环境色**：独创“时光流转”主题，背景与发光随早中晚自动推移（晨曦绿 -> 午间蓝 -> 傍晚橙 -> 夜航暗），赋予应用生命感。

### 1.2 核心差距与不足 (Cons)
1. **空间割裂感强**：页面跳转多为硬切换，缺乏共享元素动画（Shared Element Transition），未建立连贯的空间立体感。
2. **视觉噪点偏高**：卡片内部的排版留白（Whitespace）和层级不够克制，字重对比未拉开，导致在花哨的玻璃背景上信息呈现不够干脆利落。
3. **状态变更生硬**：列表加载或待办完成时，元素瞬间出现或消失，缺乏交错进场（Staggered）与平滑折叠过渡。

## 2. 核心优化行动方案

### 阶段一：视觉急救 —— 重塑卡片留白与排版层级
**目标**：提升信息获取效率，增强界面的呼吸感与高级感。
- **行动项**：
  - [x] **重构 `TodoTaskCard` 排版**：加大行距，拉开主标题（加粗深色）与副标题（细体浅灰）的对比。
  - [x] **统一圆角语言**：严格遵循外大内小的圆角曲率（如外框 `24.dp`，内框 `16.dp`），消除违和感。
  - [x] **精简冗余色块**：减少高饱和度 `Pill` 的面积与数量，让视觉焦点回归内容本身。

### 阶段二：空间魔法 —— 引入共享元素路由动画
**目标**：彻底消灭生硬的页面跳转，打造沉浸式空间感。
- **行动项**：
  - [x] **引入 `SharedTransitionLayout`**：利用 Compose 1.7 新特性包裹应用顶层。
  - [x] **首页至详情页的 Container Transform**：点击首页的“待办优先”或“今日课程”卡片时，卡片直接平滑放大扩展为详情页，返回时缩回原位。

### 阶段三：细节打磨 —— 状态反馈与交错动画
**目标**：让应用的所有动态变化如水流般顺滑。
- **行动项**：
  - [x] **列表交错进场 (Staggered Entrance)**：首页模块和日程列表在初始化时，带有微小延迟地依次向上滑入。
  - [x] **优雅的完成动效**：待办打勾时，先播放选中动画与震动，随后使用 `AnimatedVisibility` 配合 `spring` 动画平滑折叠并移除该项。
  - [x] **骨架屏过渡**：教务网络请求时，提供柔和呼吸的骨架屏而非粗暴的加载圈。

### 阶段四：感官延伸 —— 微交互、手势与无界布局 (规划中)
**目标**：打破物理屏幕的边界感，建立肌肉记忆级别的交互体验。
- **行动项**：
  - [ ] **全局物理滚动阻尼 (Overscroll)**：替换默认的生硬拉丝效果，实现类似 iOS 的橡皮筋 (Rubber Band) 滚动回弹。
  - [ ] **边缘滑动与手势驱动 (Gesture Driven)**：为详情页返回、任务右滑删除等操作加入可打断的跟手动画，让操作更具掌控感。
  - [ ] **微图标动画 (Icon Animation)**：使用 Lottie 或 Compose 绘制矢量路径动画，让底部导航栏或关键入口在点击时具备呼吸感。
  - [ ] **折叠屏与大屏自适应 (Adaptive Layout)**：不再是简单的拉伸，而是基于窗口大小类的多窗格 (Multi-Pane) 流式布局，提供平板/桌面级的沉浸体验。

### 阶段五：美学升维 —— 字体排印、光影与情感化设计 (规划中)
**目标**：从“好看好用”跨越到“极具艺术性与情感共鸣”，建立独特的视觉印记。
- **行动项**：
  - [ ] **定制化字体排印 (Typography Polish)**：引入特征鲜明的西文字体（如 SF Pro Display 或 Inter）与中文字体进行混合编排，针对大字号标题应用字距微调（Tracking/Kerning），使排版具备海报级的艺术感。
  - [ ] **多光源混合照明系统 (Multi-Lighting System)**：在现有的 Liquid Glass 基础上，为特定高优卡片（如考试倒计时、重要待办）引入环境氛围光晕（Ambient Glow）和随设备倾斜的扫光动效（Sheen Effect）。
  - [ ] **空状态情感插画 (Empty States Illustration)**：当没有待办或课表时，用轻拟物化（Neumorphism）或高级抽象几何风格的 3D 插画替换冷冰冰的文字提示。
  - [ ] **动态高斯噪点纹理 (Noise Grain Texture)**：在液态玻璃的底层轻微覆盖一层动态噪点纹理（Film Grain），打破纯数字渲染的塑料感，赋予界面更真实的物理胶片质感。

### 阶段 4：空间容器重塑与「Golden Flow」布局 (v1.2.6)
- **科学计算器彻底重构**：
    - **Golden Flow 布局**：彻底摒弃原有将键盘置于滚动列表底部的「反人类」设计。采用 `weight(1f)` 顶层容器，将计算历史与显示区域置于上方滚动，键盘永久固定于屏幕底部。
    - **Fitts's Law 优化**：大幅增加键盘按键的 Hitbox（点击热区），确保大拇指操作的精准度与效率。
    - **统一状态管线**：引入 `FocusTarget` 全局焦点路由系统，使固定键盘能无缝驱动矩阵、统计、进制转换等多个复杂模块。
    - **实时反馈**：增加 `LaunchedEffect` 驱动的实时预览引擎，用户输入过程中即时看到计算结果，无需点击等号。
    - **Z 轴浮岛 Header**：二级界面导航栏采用 Liquid Glass 浮岛设计，与主界面视觉语言达成「大一统」。

### 阶段七：跨感官与物理边界的终极形态 (未来探索)
**目标**：跳出视觉层面的“好看”，进入“通感（Synesthesia）”与“潜意识交互”的领域。
- **行动项**：
  - [ ] **UI 音效设计 (Sonic UI & Soundscapes)**：视觉和触觉之外，补全听觉。为卡片展开、任务打勾、警告等操作加入极度轻盈的定制音效（如水滴声、清脆的机械声），与马达震动（Haptics）形成 100% 同步的通感体验。
  - [ ] **流体形变与有机动画 (Fluid/Gooey Morphing)**：打破 UI 必须是几何图形（方块、圆圈）的铁律。在番茄钟计时、任务合并等场景中，采用 Metaball（元球）算法，让 UI 像真实的水滴或细胞一样进行粘连、撕裂与融合。
  - [ ] **环境变量映射 (Environmental Reflection)**：让 UI 感知真实世界。不仅是“早中晚”变色，而是接入天气与传感器：下雨时 Liquid Glass 表面出现滑落的水滴折射；用户在运动时，UI 元素自动变大以防误触。
  - [ ] **情绪化可变字体 (Variable Fonts Response)**：引入支持字重（Weight）和字宽（Width）无级渐变的字体。当考试倒计时越近，或番茄钟即将结束时，数字的字重会像心脏跳动一样逐渐加粗，用排版本身来传递“紧迫感”或“呼吸感”。

## 3. 实施准则

1. **如非必要，勿增实体**：在优化视觉层级时，优先通过增加留白和调整透明度来区分信息，尽量避免增加分割线或多余色块。
2. **动画性能至上**：所有新增动画必须通过 `remember` 缓存，避免引起不必要的 Recomposition。
3. **触觉与视觉的绝对同步**：所有的弹簧动效、展开动效的触发瞬间，必须有对应的 `HapticManager` 轻触反馈。
