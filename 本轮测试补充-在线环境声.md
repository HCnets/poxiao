# 本轮测试补充：在线环境声

1. 在番茄钟页只打开“播放白噪音”，不启动计时，是否能直接听到声音。
2. 依次切换 `山风 / 溪流 / 雨声 / 海浪 / 篝火 / 白噪音 / 粉噪`，是否都会在 1 到 2 秒内开始播放。
3. 从番茄钟页切到首页、更多、课表后，环境声是否会立刻停止。
4. 再切回番茄钟页时，如果环境声开关仍开启，声音是否会自动恢复。
5. 断网或网络较差时，打开环境声是否会提示“请检查网络后重试”，并自动关闭开关。
6. 连续切换多个环境声预设时，是否不会出现叠音、爆音或旧音源残留。
7. 首次切换到某个尚未缓存的环境声时，界面是否提示“请等待 3-5 秒完成加载”。
8. 第二次切回已经播过的环境声时，是否会明显比第一次更快。
9. 切到 `粉噪` 时，界面是否提示已自动降低音量，且实际响度是否明显低于之前版本。
